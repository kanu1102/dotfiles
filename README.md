# dotfiles
settings like a sh*t! lol
* .vimrc
* .zshrc

## Install neobundle
```
$ mkdir -p ~/.vim/bundle
$ git clone https://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim
$ vim
```


## clone
```
$ cd ~/
$ git clone git@github.com:kanu1102/dotfiles.git
```

## synbolic link
```
$ ln -s dotfiles/.vimrc .vimrc
$ ln -s dotfiles/.zshrc .zshrc
```
